LegoMinigolfV1
==============

Play Minigolf with LegoMindstorms


This is a project from Hochschule Pforzheim, Germany

Das ganze Projekt wird im Rahmen einer Projektarbeit bearbeitet.


Die Programmiersprache für den NXT ist RobotC.
